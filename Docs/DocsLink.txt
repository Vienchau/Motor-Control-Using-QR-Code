Khi compile và run code trong Qt, cần chú ý việc các file trong folder "bin" (file .dll) có đúng với mingw hay không.
Việc chọn mingw phiên bản nào có ảnh hưởng tới việc run code, khi run sẽ xảy ra lỗi "cant located ...dll". 
Chú ý lúc gen code bằng CMake thì mình đã gen phiên bản nào để chọn phiên bản Mingw32 phù hợp trong Qt (Hoặc
các trình code sau này).
